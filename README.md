# markdown_note_taking_app

**markdown_note_taking_app** is a simple yet powerful note-taking application that supports real-time Markdown preview. Built using Python and Tkinter, this app allows users to write and preview Markdown syntax instantly, making it an ideal tool for note-taking, blogging, or writing documentation.

## Features

- **Markdown Editor**: A real-time, interactive editor built with Tkinter.
- **Markdown Preview**: Live preview of the Markdown content as HTML.
- **Real-Time Rendering**: Instantly see the rendered Markdown as you type.
- **Save and Load Notes**: Save your notes as `.md` files and load them later.
- **Cross-Platform**: Built with Python and Tkinter, making it compatible across multiple operating systems.

## Installation

To run the **markdown_note_taking_app**, you need Python 3.x and the `markdown` library installed.

1. **Clone the repository**:

   ```bash
   git clone https://github.com/DeveloperJarvis/markdown_note_taking_app.git
   cd markdown_note_taking_app
   ```

2. **Install dependencies**:

   ```bash
   pip install -r requirements.txt
   ```

3. **Run unit tests**:

   ```bash
   python -m unittest test_markdown.py
   ```

## Usage

### Running the Application

1. Clone the repository or download the source code.
2. Open a terminal and navigate to the project directory.
3. Run the app using Python:

   ```bash
   python main.py
   ```

The application will open a window with:

- A Markdown editor on the left.
- A real-time rendered preview on the right.

### Features in the App:

- **Markdown Editing**: Type your Markdown content in the left text box.
- **Preview**: The rendered content is shown on the right as you type, thanks to the real-time update mechanism.
- **Save Notes**: You can save your work in a `.md` file and load it later for editing.
